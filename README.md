🐳 DevOps Калькулятор

Мой pet-проект для портфолио. Это не просто калькулятор, а полноценный многосервисный DevOps-стек, который демонстрирует навыки контейнеризации, работы с базами данных и CI/CD.



- Python 3.11 — логика калькулятора  
- PostgreSQL 15 — хранение истории вычислений  
- Docker & Docker Compose — контейнеризация и оркестрация  
- GitHub Actions — автоматическая сборка при пуше
